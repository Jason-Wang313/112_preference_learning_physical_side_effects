# Hostile Prior Work

The broad claim "human preferences can improve robot behavior" is not novel.

Relevant pressure points:
- Preference-based reinforcement learning and RLHF already learn from comparisons.
- Constraint learning and safe robot learning already model undesirable outcomes.
- Human-robot collaboration work already studies assistance preferences.
- Active preference learning already queries informative comparisons.

The novelty boundary is narrow: this paper tests whether explicit physical side-effect mechanisms improve preference learning beyond generic preference and side-effect classifier baselines.
